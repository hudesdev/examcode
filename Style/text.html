todo @font-face

p{ font-family:Arial,'Times New Roman',Webdings; }

This CSS code renders the fonts in order from left to right until it finds one that is available on the client computer. If the font name contains spaces, it must be contained within quotes. If none of the specified fonts are available, the text falls back to the browserâ€™s default font. In the previous example, the client looks first for the Arial font. If that is not installed, it then looks for the Times New Roman font, and so on. This is a simple approach, but many people prefer to use fonts that are not available on the system. There are many custom fonts available on the Internet for inclusion in your web applications. These fonts are known as WOFF (Web Open Font Format). To use these fonts in your webpage, you define a font family using the special keyword @font-face.

The following code defines a @font-face for a webpage and implements it for the elements of the page:

@font-face {

font-family: "My Nicer Font";

src: url('fonts/my_woff_font.eot');

src: url('fonts/my_woff_font.woff');

}

p { font-family: 'My Nicer Font'; }
